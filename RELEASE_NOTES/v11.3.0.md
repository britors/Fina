# Fina v11.3.0

## Versão

- Atualiza a versão do produto para a nova linha 11.3.0.

## Base da entrega

- Open Finance com credenciais para Pluggy, Belvo e Klavi.
- Sincronização inicial via Pluggy para contas, saldos e lançamentos.
- Lista de modelos por provedor em `Configurações > IA`.
- Parcelamento de lançamentos em cartão de crédito.
- Filtro de mês aprimorado em orçamento.
- Cards de vales mostrando o disponível para gastar.
