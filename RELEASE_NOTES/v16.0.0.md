# Fina v16.0.0

## Versão

- Abre a linha 16.0 com a base operacional de Pix via Open Finance.

## Pix

- Nova tela `Pix` no menu lateral.
- Histórico local de tentativas Pix com filtros por status e período.
- Detalhes de auditoria com provedor, conta de origem, chave minimizada, ID externo, erro e lançamento vinculado.
- Cadastro de favorecidos Pix com nome, chave, instituição e observações.
- Validação local de CPF, CNPJ, e-mail, telefone e chave aleatória.
- Fluxo inicial `Novo Pix` com revisão explícita e registro de tentativa em modo sandbox/auditoria.

## Banco de dados

- Nova tabela `pix_payments` para histórico e auditoria de tentativas Pix.
- Nova tabela `pix_recipients` para favorecidos Pix salvos localmente.
